# Unia Websites

Unia 系列网站、管理平台、机器人及相关服务的统一项目入口。

## 子项目

| 项目 | 说明 | 仓库 |
| --- | --- | --- |
| Unia-Admin-Bot | Unia 管理机器人及后台服务 | [GitHub](https://github.com/Polaris-Leo/Unia-Admin-Bot) |
| Unia-Danmuku | Unia 弹幕相关服务 | [GitHub](https://github.com/Polaris-Leo/Unia-Danmuku) |
| Unia-BiliBot | Unia 哔哩哔哩机器人 | [GitHub](https://github.com/Polaris-Leo/Unia-BiliBot) |
| Unia-Admin-APP | Unia 管理客户端 | [GitHub](https://github.com/Polaris-Leo/Unia-Admin-APP) |
| Unia-BiliCookie | 哔哩哔哩 Cookie 管理服务 | [GitHub](https://github.com/Polaris-Leo/Unia-BiliCookie) |

## 克隆项目

主仓库使用 Git Submodule 管理子项目。

```bash
git clone --recurse-submodules https://github.com/Polaris-Leo/Unia-Websites.git
```
